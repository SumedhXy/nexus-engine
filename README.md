# 🛡️ NEXUS: Emergency Orchestration Engine 🚀

**NEXUS** is a high-fidelity, professional emergency command platform designed for real-world crisis orchestration. It leverages **Gemini-2.0-Flash** AI to perceive natural language distress signals and autonomously coordinate secondary and tertiary responder dispatches with a strict **Life-Safety First** priority.

---

### 🏛️ CORE ARCHITECTURE
1.  **🧬 PERCEPTION LAYER**: High-intensity AI classification with a 4-second "Quota Shield" and a SENTRY Local Heuristic fallback.
2.  **🚑 ORCHESTRATION HUB**: Tactical SOP Engine that automatically prioritizes **Ambulance** and **Hospital** units as the primary responders for all Civil and Traffic emergencies.
3.  **📡 BROADCAST GRID**: 200% Mission-Redundancy via **Twilio SMS** and **Telegram Bot API**, transmitting live GPS fixes and Google Maps links instantly.
4.  **🔐 IDENTITY HANDSHAKE**: Production-hardened security with **JWT Token Verification** and **SlowAPI Rate Limiting** to prevent tactical interference.
5.  **🏛️ MISSION LOGS**: Persistent cloud archival of all past incidents via **Firebase Realtime Database**.

---

### 🚀 TACTICAL QUICKSTART (LOCAL)

```powershell
# 1. INSTALL MISSION LIBRARIES
pip install -r requirements.txt

# 2. SEIZE CONTROL (Launch Engine)
python -m uvicorn main:app --host 0.0.0.0 --port 8888 --reload

# 3. LAUNCH DASHBOARD
npm run dev
```
